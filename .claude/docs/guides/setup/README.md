# Setup and Configuration Guides

Guides for project setup, configuration analysis, and standards extraction.

## Purpose

This directory contains guides for the /setup command and related configuration management tasks. Use these guides when initializing projects, analyzing CLAUDE.md compliance, or extracting standards from existing documentation.

## Guide Index

### setup-modes-detailed.md
**Topic**: Detailed documentation of /setup command modes
**Audience**: Developers using or extending /setup command
**Prerequisites**: Understanding of CLAUDE.md structure

### standards-analysis.md
**Topic**: Standards extraction and analysis algorithms
**Audience**: Developers working on standards detection
**Prerequisites**: Setup modes

### extraction-strategies.md
**Topic**: Content extraction strategies for standards
**Audience**: Developers implementing extraction features
**Prerequisites**: Standards analysis

### bloat-detection.md
**Topic**: Bloat detection and cleanup recommendations
**Audience**: Developers maintaining CLAUDE.md files
**Prerequisites**: Setup modes

## Reading Order

Suggested sequence for learning setup system:
1. setup-modes-detailed.md
