# Templates

This directory contains reusable file templates for creating new commands, agents, and other components.

## Overview

Templates provide consistent starting points for new files, ensuring they follow established patterns and include all required sections.

## Available Templates

| Template | Description |
|----------|-------------|
| [_template-bash-block.md](_template-bash-block.md) | Template for bash block structures |
| [_template-command-guide.md](_template-command-guide.md) | Template for command documentation |
| [_template-executable-command.md](_template-executable-command.md) | Template for executable commands |

## Usage

1. Copy the appropriate template
2. Rename to match your component name
3. Replace placeholder content with actual implementation
4. Remove any unused sections

## Template Conventions

- Template filenames start with underscore prefix (`_template-`)
- Templates include placeholder comments indicating required content
- All sections are pre-formatted to meet documentation standards
