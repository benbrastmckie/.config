# Compliance Tests

## Purpose

Tests for code standards and compliance requirements. This directory contains tests that verify adherence to coding standards, including bash error handling patterns, argument capture, history expansion safety, and compliance remediation validation.

## Test Files

| File | Description |
|------|-------------|
| `test_agent_validation.sh` | Test agent file validation |
| `test_argument_capture.sh` | Test CLI argument handling |
| `test_bash_command_fixes.sh` | Test bash command fixes |
| `test_bash_error_compliance.sh` | Test bash error handling |
| `test_bash_error_integration.sh` | Test bash error integration |
| `test_compliance_remediation_phase7.sh` | Test compliance remediation |
| `test_history_expansion.sh` | Test bash history expansion |
| `test_no_if_negation_patterns.sh` | Test if negation patterns |

## Running Tests

```bash
bash tests/features/compliance/test_bash_error_compliance.sh
```
